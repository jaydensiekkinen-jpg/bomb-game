# bomb-game
a game about defusing a bomb
made with a breadboard speaker leds and alot of wires in difrent colors for more fun in the defusing proses!
play with family or freinds but connect the correct pins to the speakers and leds 
Made in ardorino IDE 
HOW IT WORKS
the game starts and the game beggins 
now be quick analyse the wires 
cut or pull out the right one!
if you do it correctly it will stop beeping 

Hardware used  ardorino uno
breadboard
LEDs
buzzer
wires
"optional" reset button

AI was used to set up my devlogs thanks to gemini but the games code was my own work!
